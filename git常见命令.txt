git init 初始化git仓库
git branch 查看分支
git branch new_branch(创建新分支
git add  将文件添加到暂存区
git add . 将当前目录下所有修改和新增的文件添加到暂存区
git add -u 只将已被跟踪的文件中修改或删除的内容添加到暂存区，不包括新增文件
git add -A 将所有修改、新增的文件添加到暂存区
git commit -m "描述" 将add的文件添加到本地仓库，   -m是使用描述的意思
git commit -a 将所有已被追踪文件中修改和删除的内容直接提交到本地仓库，跳过git add
git commit --amend 修改上一次的提交记录
git status 查看状态
git status -s：以简洁的格式显示文件状态，只显示文件的状态标识和文件名
git config --global user.name "设置名称"
git config --global user.emali "设置邮箱"
git rm --cached "文件名"   取消追踪、
git reset  “文件名”  从暂存区移除文件
git reset "代码提交的哈希值"  前七位，回退版本
git log 查看提交记录
git log --since="2025-12-08" --until="2025-12-08"  按时间筛选
git log --author="your  name "按名称筛选
git checkout "不同分支" 切换分支
git checkout -b "分支名"   创建并切换
git remote  add "仓库名"  "仓库地址"   连接远程仓库
git clone   "仓库地址"    拉取仓库到本地
git remote -v 显示所有已添加的远程仓库的别名和地址
git push "仓库名称" "仓库地址"(已连接可以不用写）  推送内容至仓库
git merge  合并分支
