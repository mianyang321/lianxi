git init 初始化git仓库
git branch 查看分支
git branch new_branch(创建新分支
git add  将文件添加到暂存区
git add . 将当前目录下所有修改和新增的文件添加到暂存区
git add -u 只将已被跟踪的文件中修改或删除的内容添加到暂存区，不包括新增文件
git add -A 将所有修改、新增的文件添加到暂存区
git commit -m "描述" 将add的文件添加到本地仓库，   -m是使用描述的意思
git commit -a 将所有已被追踪文件中修改和删除的内容直接提交到本地仓库，跳过git add
git commit --amend 修改上一次的提交记录
git status 查看状态
git status -s：以简洁的格式显示文件状态，只显示文件的状态标识和文件名
git config --global user.name "设置名称"
git config --global user.emali "设置邮箱"
git rm --cached "文件名"   取消追踪
